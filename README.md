## Food API

Very simple REST API for foods to practice making an API using Express, Node.js and MongoDb Atlas.


**Functionality:**
- Get all menu items 

- Add a menu item

- Update/Delete a menu item by Id

- Get menu items by types (Ex: desserts, drinks, etc)
